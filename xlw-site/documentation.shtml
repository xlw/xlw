Documentation

The documentation is included with the installer for the XLW source code, and may also be downloaded separately - visit the downloads page on SourceForge. Alternatively the documentation may be browsed online at the links below.

Reference Manual

The reference manual is autogenerated by the doxygen utility from comments in the source code, and documents all of the classes in XLW.


http://xlw.sourceforge.net/refman/

Design Document for XLW Version 2

Includes a description of the InterfaceGenerator module which parses the header files of your application and autogenerates the source code for the corresponding XLW Excel addin. All of the information remains relevant for version 3, and the document has been updated to reflect changes in path names etc.


http://xlw.sourceforge.net/version2/

Design Document for XLW Version 3

Summarizes enhancements made to XLW in support of new features in Excel 2007. This document is included in the reference manual.


http://xlw.sourceforge.net/refman/version3.html

Getting Started Document for XLW Version 4 (Beta 2)

A quick introduction and getting started guide to XLW 4. This document is designed to let the user very quickly get going with the latest Beta release of v4.0.0


Doc-4.0.0b2.TXT
